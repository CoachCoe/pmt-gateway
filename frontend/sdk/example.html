PMT Gateway SDK

Polkadot Payment Gateway - JavaScript SDK Demo

1. Initialize SDK

First, initialize the PMT Gateway SDK with your API key:

// Initialize PMT Gateway const pmt = PMTGateway.create({ publicKey: 'pk_test_your_api_key_here', environment: 'test', debug: true });

2. Create Payment Intent

Create a payment intent for $25.00 USD:

// Create payment intent const paymentIntent = await pmt.createPaymentIntent({ amount: 2500, // $25.00 in cents currency: 'usd', crypto_currency: 'dot', metadata: { order_id: 'order_123', customer_email: 'customer@example.com' } });

3. Payment Widget

Display the payment widget for customers to complete payment:

// Create payment widget const widget = pmt.createPaymentWidget({ container: '#payment-widget', paymentIntent: paymentIntent, onSuccess: (event) => { console.log('Payment successful:', event); }, onError: (error) => { console.error('Payment failed:', error); }, theme: { primaryColor: '#6366f1', borderRadius: '8px' } });

4. Wallet Management

Manage wallet connections and authentication:

// Get supported wallets const wallets = await pmt.getSupportedWallets(); // Connect to a wallet const connection = await pmt.connectWallet('polkadot-js'); // Authenticate wallet const token = await pmt.authenticateWallet('merchant_id');
